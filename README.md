# Sharpn: History

Sharpen your knowledge of world history.

An AI-powered history learning app with structured curricula, interactive lessons, and quizzes — powered by Claude.

## Setup

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```
ANTHROPIC_API_KEY=your-api-key
```

## Run

```bash
python app.py
```

Open [http://localhost:5000](http://localhost:5000)
